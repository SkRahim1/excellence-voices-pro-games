# Excellence Voices Pro: Visual UI Design Showcase

This document presents premium, high-fidelity UI design mockups for the student dashboards, AI voice rooms, and teacher portals.

---

## 1. Student Dashboard (Teen Dark-Mode Theme)
This mockup represents the dashboard interface presented to students age $\ge 11$. It features a modern, cyberpunk dark aesthetic designed to appeal to middle and high school students.

* **Design Elements:** Translucent cards (glassmorphism), neon progress indicators, daily time-goals tracking calendar (Mon-Fri), and class leaderboard sidebar.

## 2. AI Voice Tutor (Kids Theme Interface)
This mockup shows the gameplay screen for younger students (ages < 11). The UI shifts to a soft, engaging, bright aesthetic featuring friendly characters and speech bubbles to encourage active participation.

* **Design Elements:** "Voicey the Owl" AI Companion, large bubble text cues, wave visualizer for amplitude feedback, and circular microphone action controls.

## 3. Teacher Hub & Analytics Dashboard
Designed with a clean, light interface for classroom instructors and school principals, optimizing readability and administrative workflows.

* **Design Elements:** Top KPI widgets for classroom stats, sorting progress matrices, structured homework creation panel, and a sidebar for listening to and grading student voice records.
